# Hotel Nebula — Banco de Dados

Este projeto tem como objetivo modelar e implementar um banco de dados para o sistema do **Hotel Nebula**, organizando informações de hóspedes, quartos, reservas, hospedagens, pagamentos, funcionários, serviços e avaliações.

---

## Objetivo

Estruturar um banco de dados relacional para melhorar a gestão do hotel, permitindo controle de reservas, hospedagens, serviços e análises operacionais.

---

## Estrutura do Banco

O sistema contém as seguintes tabelas:

* **hospedes** → dados dos clientes do hotel
* **quartos** → informações dos quartos disponíveis
* **reservas** → controle de reservas feitas pelos hóspedes
* **hospedagens** → registro de estadias realizadas
* **pagamentos** → registros de pagamentos das hospedagens
* **funcionarios** → funcionários do hotel
* **servicos** → serviços extras oferecidos
* **feedbacks** → avaliações dos hóspedes

---

## Tecnologias

* MySQL
* Modelagem relacional
* Linguagem SQL

---

## Consultas principais

O sistema permite responder perguntas como:

* Quartos disponíveis em determinado período
* Hóspedes com mais reservas
* Faturamento total do hotel
* Serviços mais consumidos
* Avaliações dos quartos
* Reservas canceladas
* Serviços realizados por funcionário

---

## Observação

Este projeto foi desenvolvido com foco educacional, simulando um sistema real de gestão hoteleira.
